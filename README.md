
# PyMySQL-CLI

Interactive console for the PyMySQL package in python.



## Installation

do:

```
  pip install -r requirements.txt
  python main.py
```

or

```
  pip3 install -r requirements.txt
  python3 main.py
```

done!




## Features

- SHOWTABLES
- SHOWDATABASES
- SETCOMMAND (to execute custom payload)
- Q <COMMAND> (to execute a custom payload quickly)
